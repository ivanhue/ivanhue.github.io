YouTube to Spotify Playlist Converter

YouTube to Spotify Playlist Converter is a Python command-line tool that automatically converts YouTube Music playlists into Spotify playlists. It demonstrates my skills in API integration, and Python scripting. The tool searches and matches songs, creates new playlists in Spotify, and shows progress with clear messages.

Web Data Scraper & Dataset Manager

This project automatically extracts information from websites to create and update a dataset. It uses Python, asyncio, and httpx for fast and efficient web scraping, and regex to filter relevant pages. The tool works in two phases: first it collects data from the initial set of websites, then it searches for additional related pages. The project demonstrates skills in data extraction, API handling, automation, and Python scripting.
